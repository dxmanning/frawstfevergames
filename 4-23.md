# Frawst Fever Games — Work Report (2026-04-23)

## 1. Shipping Integration — What You Need to Connect

For Canada-only shipping, here are the options, ranked by ease:

### Option A — Stripe Shipping (easiest, already partially set up)
Stripe is already integrated. It can collect shipping addresses and optionally use **Stripe Tax** + **Shipping Rates** — but doesn't buy labels. You manually ship + enter tracking.
- **Connect to:** [dashboard.stripe.com](https://dashboard.stripe.com) → Products → Shipping rates
- **Need:** Nothing extra — already connected

### Option B — Canada Post (most popular for small Canadian shops)
- **Connect to:** <https://www.canadapost-postescanada.ca/cpc/en/business/shipping-services/small-business.page>
- **Need:**
  - Canadian business (or personal) Canada Post account
  - Register for the **Developer Program** → get API Key + API Secret
  - Contract number (Solutions for Small Business is free)
  - Credit card on file for buying/printing labels

### Option C — Shippo (multi-carrier aggregator, recommended)
One API gets you Canada Post + UPS + Purolator + FedEx rates and labels in one place. Easier than going direct.
- **Connect to:** <https://goshippo.com/sign-up>
- **Need:**
  - Email signup (no business verification needed to start)
  - Add your pickup address
  - API token (free tier: pay per label, no monthly fee)

### Option D — Easyship / EasyPost (similar to Shippo)
- <https://www.easyship.com> or <https://www.easypost.com>

### Recommendation
Start with **Shippo**. Sign up with email, get an API key in 5 minutes, fetch live Canada Post + UPS rates. We can wire the current weight-based `shipping.ts` estimate to call Shippo's rates API at checkout so customers see real carrier rates.

**What to provide when ready:**
- `SHIPPO_API_KEY` (or `CANADA_POST_USERNAME` + `CANADA_POST_PASSWORD` if going direct)
- Your pickup address (city, postal code) so rates calculate from the right origin
